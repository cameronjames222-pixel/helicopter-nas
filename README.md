# helicopter-nas
The Story Behind This Project

During a heavy week of commercial helicopter ground school, I found myself completely overwhelmed trying to track the endless cloud clearances, equipment rules, and unique Class G helicopter exemptions. The data was blurring together, and the standard, dense text walls in the regulations weren't sticking.

In a sudden brainwave, I realized the best way to force myself to truly master the material was to build the exact tool I wished I had.

This interactive visualizer is the result of that deep-dive study session. It strips away the clutter to map standard FAA airspace rules directly to the low-and-slow reality of helicopter operations. It was born out of a personal need to build bulletproof judgment for my own ratings, and it's shared here as an interactive, precise reference for fellow helicopter pilots. Built by a pilot, for pilots.
